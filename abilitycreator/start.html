Ability Creator Help

This section is made to help people understand what all the fields mean within the Ability Creator. If things still aren't clear after reading this, contact StrangeOne101.

Basic Fields

Name: Name of the ability. Cannot have a space and is the name of the ability users type when going /bending bind <ability>

Author: Your name. This will tell everyone who made the ability! E.g. StrageOne101

Version: The version of the ability. This is for when you do updates on your ability. When you update it, increase it by 0.1 or something so users know that it's a different version!

Help Info: This tells people how to use your move. Everything in this field will be shown in /bending help <ability>

Element: What element this ability is. Pretty self explanatory.

SubElement: What subelement this ability is. Also pretty self explanatory.

Cooldown: When people successfully use this move, how long should they not be able to use it? This is in seconds!

Move Type: What kind of move this is. Keep this on block/particle for now (no other moves supported)

Move Start: How your move is activated. E.g. "Click" for moves like FireBlast, "Shift" for moves like Blaze

Source Selection: How people select a source. NOTE: Moves that don't require sources will start from the selected location (think of AirBlast)

Speed: The amount of blocks per second the ability moves at.

Range: The number of blocks until the ability dissipates.

Redirect: How the move can be redirected. Constant redirection means it will follow where the player is looking.

Damage: The amount of damage this move does (in hearts).

Max Hits: Like the config option in torrent that lets admins choose how many times the move hits before it won't hit anymore (from one blast).

Death Message: The message that shows when players are killed by the ability.

Display: How this move looks. Use the builder button on the right to configure this!

Advanced Fields

Collision Radius: The radius (in blocks) that mobs and players can be hit in.

Package Name: A field for real developers. This field will do nothing in game and is just an option for a slight change in the code.
