Ability Info

This is where the main information for you ability goes. The "Help Text" is the text users see when running the /bending help command with your ability.

General

For stats that can be modified on the move, such as the element it is and the cooldown it gives.

Info

Cooldown: Once the move is used, how long until they can use it again. Measured in seconds.
Move Start: What triggers the begging of a move. For a move like FireBlast, you'd use "Click"
Move Speed: How many blocks per second the move travels.
Range: How far the move can go before it dissipates. Measured in blocks.
Redirect: If the move should allow redirects. "Never" means it can't be redirected and "Constantly" means it will follow your cursor (like Torrent)

Require Avatar: Whether the user needs to be an avatar OR if it's just a move that doesn't belong to any element. If this is the case, set to "No". (Only an option for avatar moves)

Run Java Code

Allows you to run java code. Please note that although it checks syntax for you, it will not show errors.

Create Explosion

Creates an explosion at the ability's current location.




Info

Explosion Radius: The size of the explosion. For reference, TNT is 4.
Destroy Blocks: If the explosion should damage surrounding blocks. If false, it will just damage mobs/players.
Create Fire: If the explosion should create fire blocks when it explodes. Similar to ghast fireballs.
